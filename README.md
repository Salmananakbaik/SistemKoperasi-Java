# Sistem Informasi Koperasi

Sistem Informasi Koperasi merupakan aplikasi desktop yang dikembangkan menggunakan Java NetBeans dan MySQL untuk membantu pengelolaan data koperasi secara terkomputerisasi.

## Fitur
- Login Admin
- Dashboard
- Data Anggota
- Data Simpanan
- Data Pinjaman
- Laporan

## Teknologi
- Java
- NetBeans IDE
- MySQL
- JDBC

## Struktur Project
```
src/
nbproject/
build.xml
manifest.mf
```

## Cara Menjalankan
1. Clone repository ini.
2. Import project ke NetBeans.
3. Import database MySQL.
4. Tambahkan MySQL Connector/J.
5. Jalankan aplikasi.
